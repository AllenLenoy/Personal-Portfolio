## ✨ Features

- **📺 Cinematic Design Language**: A "Premium Dark" aesthetic inspired by modern streaming interfaces and high-end automotive configurators.
- **🌀 3D Coverflow Carousels**: Custom-engineered 3D transforms for Projects and Certifications, featuring glassmorphism and depth-of-field effects.
- **🧭 Smart Navigation**: A persistent header with real-time IntersectionObserver scroll tracking and a unique theme-aware bento-nav.
- **⚡ Peak Performance**: Built on **Next.js 16 (Turbopack)** and **React 19** for instantaneous page transitions and optimized server-side rendering.
- **🛠️ Interactive Control Hub**: A draggable "Floating Hub" providing quick access to socials, theme toggling, and global actions.
- **🍱 Bento-Grid Skills**: A modern grid layout for technical assets with dynamic iconography and hover-state highlights.
- **🖱️ Magnetic Micro-interactions**: Physics-based magnetic effects on critical UI elements (buttons, nav links) for enhanced tactile feedback.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AllenLenoy/Personal-Portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
